# 🎟️ Event Booking Platform (MERN Stack)

A full-stack event booking web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). This platform allows users to explore events, book tickets securely using email OTP verification, and track their bookings through a personalized dashboard.

It also provides a powerful admin panel for event organizers to manage events, bookings, and analytics.

---

## 🚀 Features

### 👤 User Features
- 🔍 Browse available events
- 🎫 Request ticket bookings
- 📧 Secure email OTP verification for authentication
- 📊 Personal dashboard to track booking status
- 🕒 View pending, confirmed, and rejected bookings

### 🛠️ Admin Features
- ➕ Create and manage events
- 📋 View and manage booking requests
- 💰 Confirm payments
- 📈 Analytics dashboard:
  - Total revenue
  - Confirmed bookings
  - Pending requests

---

## 🧠 Key Concepts Used

- MERN Stack Architecture
- Authentication with Email OTP
- Role-Based Access Control (User/Admin)
- RESTful APIs
- Secure Booking Logic
- State Management (React)
- Dashboard UI & Data Visualization
